Is this safe?

  • lock

    Your headers are stored only in your browser session — never written to a database or disk.

  • person_off

    Nobody — including the developer — can see your credentials. The server reads them per-request and discards them immediately.

  • logout

    Closing your browser clears the session completely.

  • code

    Don't take our word for it — the full source code is open on GitHub.
